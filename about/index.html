自学课程代码项目

人工智能 / 寻路

寻路是在平面上的两个点之间找到最优路径的问题。寻路的最明显用法可能是在计算机游戏中，当一个物体被指示从其当前位置移动到目标位置，同时沿途避开障碍物（例如墙壁，敌人的火力）时，分别对BFS (Breadth-First Search )、UCS (Uniform-Cost Search)和A*搜索进行了实现。

实现的语言版本：Java、C++、 Python。

人工智能 / 决策树学习

实现的语言版本：Java、C++、 Python。

算法与数据结构分析 / 大数运算

对大数运算中的加（School addition）、减（School subtraction）、乘（Karatsuba multiplication）进行实现。

实现的语言版本：C++。

计算机网络与应用 / 可靠传输协议

基于案例代码进行实现。

实现的语言版本：C。

计算机网络与应用 / 路由算法

对距离向量（Distance Vector）、毒性反转（Poisoned Reverse）算进行实现。

实现的语言版本：Python、Java、C++。

计算机视觉 / 感知机、多分类器、反向传播

感知机是对线性可分的2维数据进行简单实现。多分类器是在感知机的基础上进行复合形成3分类器。反向传播是对forward、backward、loss的推导理解并实现，最后使用Keras对某个银行假账数据的分析，precision为88.32%。

实现的语言版本：Python。

分布式系统 / 基于Java RMI的栈操作

理解Java中的远程调用是如何工作的，然后使用RMI对一个栈结构数据进行push、pop、isEmpty、delayPop运算的实现。

实现的语言版本：Java。

分布式系统 / 汇总和分发ATOM feeds

构建一个客户端/服务端系统，对ATOM feeds进行汇总和分发。实现功能点有：对ATOM XML构建与解析（无第三方包）、基于Socket实现一个简单的HTTP client/server、基于LamportClock对分布式请求时序进行管理。

实现的语言版本：Java。

分布式系统 / 基于Paxos的议会主席选举

理解Paxos并实现一个议会主席选举系统。该系统模拟了4种网络情况的用户（及时响应、传播延时高、响应不及时、正常网络），由1-2名用户同时提出主席申请来确定最终主席。

实现的语言版本：Java。

统计机器学习导论 / SVM实现

从零开始的SVM实现（除cvxopt以外，凸优化求解），分别对原始形式（primal）和对偶形式（dual）进行实现。

实现的语言版本：Python。

大数据挖掘 / Map-Reduce简单应用
